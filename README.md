# Camera Calibration & Distortion Correction
OpenCV를 이용하여 카메라의 렌즈 왜곡을 측정하고, 이를 보정하여 평면 이미지를 얻는 프로그램</br>
체스보드 패턴을 활용한 calibrateCamera 알고리즘과 initUndistortRectifyMap을 조합하여 구현</br>
이미지 선택 과정, 보정 전/후 비교 영상, 보정 후 영상 자동 저장 지원

## 기능
1. 영상 전처리 및 해상도 최적화
   - 입력받은 비디오 파일을 가로 800px(TW) 기준으로 비율을 유지하며 리사이즈
   - 처리 효율성을 높이고 보정 전/후 프레임의 일관성 유지

2. 인터랙티브 이미지 선택 (Calibration Data)
   - 영상 재생 중 사용자 입력을 통한 정밀한 학습 데이터 수집
   - Space(일시정지): findChessboardCorners를 통해 체스보드 코너를 실시간으로 탐색 및 시각화
   - Enter(이미지 선택): 코너 검출이 확인된 프레임을 캘리브레이션 데이터 리스트에 추가 후 재생 재개
   - 일시정지 진행 순간도 기록하여 작업 과정을 영상으로 보존

3. 정밀 카메라 캘리브레이션
   - 수집된 체스보드 프레임을 바탕으로 카메라 행렬(K)과 왜곡 계수(Distortion Coefficients) 산출
   - 실제 세계의 좌표(3D)와 이미지 상의 좌표(2D)를 매핑하여 RMS Error 및 보정 파라미터 출력

4. 왜곡 보정 및 결과 시각화
   - Remap 보정: 산출된 데이터를 기반으로 렌즈에 의한 기하학적 왜곡(배럴/핀쿠션 왜곡)을 실시간 보정
   - 백그라운드 자동 저장: 보정된 순수 결과 영상을 무인 모드로 먼저 저장하여 데이터 무결성 확보
   - 실시간 비교 출력: 보정 전(Original)과 보정 후(Rectified)를 Tab 키로 전환하며 시각적 차이 확인 가능

5. 다중 출력 자동 저장
   - 프로그램 종료 시 현재 날짜와 시간을 포함한 3가지 형식의 영상 자동 저장
     + Play_파일명_날짜.avi: 사용자가 이미지를 선택하는 전 과정 기록
     + Result_파일명_날짜.avi: 화면 출력 없이 보정만 완료된 순수 결과 영상
     + Result_Play_파일명_날짜.avi: 보정 전/후 비교 확인 과정 기록

### 조작 방법
- 이미지 선택 단계 (Calibration)
  + Space : 영상 일시정지 및 체스보드 코너 표시
  + Enter : (일시정지 상태에서) 현재 프레임 선택 및 재생 재개
  + ESC : 이미지 선택 종료 및 캘리브레이션 시작

- 왜곡 보정 확인 단계 (Correction)
  + Tab : 보정 전(Original) / 보정 후(Rectified) 모드 전환
  + Space : 일시정지
  + ESC : 프로그램 종료 및 리소스 해제
